Qubits and Kets

Qubits are normally written in the so-called braâ€“ket notation. Each qubit is represented as a ket, of the form $\ket{a}$. For instance, the computational basis states are written as $\ket{0}$ and $\ket{1}$, and called ket 0 and ket 1, respectively. Thus, any qubit can be described by a linear combination of these two basis states as $\ket{a} =\alpha_0\ket{0} +\alpha_1 \ket{1}$, where $\alpha_0, \alpha_1 \in \mathbb{C}$ such that $ |\alpha_0|^{2}+|\alpha_1|^{2}=1$.
For any qubit $\ket{a}$, the following notation is used: \[a_{0}=\braket{a|0}\text{ and }a_{1}=\braket{a|1},\]
so the qubit $\ket{a}$ can be write by: \begin{equation} \label{eq:qubit-coordenadas} \ket{a}=a_0\ket{0}+a_1\ket{1} \end{equation} For other basis, like for example $\set{\ket{+}, \ket{-}}$, where $\ket{+}=\frac{\ket{0}+\ket{1}}{2}$ and $\ket{-}=\frac{\ket{0}-\ket{1}}{2}$, then a qubit $\ket{a}$ can be described like: \begin{equation*} \ket{a}=a_+\ket{+}+a_-\ket{-} \end{equation*} In general, for $n$ qubits $\ket{a_1}\dots\ket{a_n}$, the composite ket $\ket{a}=\ket{a_1\cdots a_n}$ can be written as: \begin{equation} \ket{a}=a_{k0}\ket{a_1\cdots a_{k-1}0a_{k+1}\cdots a_n}+a_{k1}\ket{a_1\cdots a_{k-1}1a_{k+1}\cdots a_n}. \end{equation} For example, for two qubits $\ket{a}$ and $\ket{b}$, the following can be written: \begin{align} \ket{ab}&=a_0\ket{0b}+a_1\ket{1b}\\ &=b_0\ket{a0}+b_1\ket{a1}\\ &=a_0 b_0\ket{00}+a_0 b_1\ket{01}+a_1 b_1\ket{10}+a_1 b_1\ket{11} \end{align}
For any ket $\ket{a}=\ket{a_1\dots a_n}$ of $n$ qubits, the following notation is used: \begin{equation} \label{eq:nqubit-only-one-state} \ket{a}^{k}_{\gamma} = \ket{a_1\cdots a_{k-1}\gamma a_{k+1}\cdots a_n}, \end{equation} where $k \in \mathbb{N}_n$ and $\gamma\in\{0,1\}$.
Consequently, we can write the previus ket as follows: \begin{equation} \ket{a}= a_{k0}\ket{a}^{k}_{0}+a_{k1}\ket{a}^{k}_{1}. \end{equation} For instance, if $\ket{a_1},\ket{a_2}$ and $\ket{a_3}$ are three qubits, then:
  • $\ket{a_1}^{1}_{0}=\ket{0}$ and $\ket{a_1}^{1}_{1}=\ket{1}$.
  • $\ket{a_1 a_2}^{2}_{0}=\ket{a_1 0}$.
  • $\ket{a_1 a_2 a_3}=a_{20}\ket{a_1 0 a_3}+a_{21}\ket{a_1 1 a_3}=a_{20}\ket{a}^{2}_{0}+a _{21}\ket{a}^{2}_{1}$.
  • $\ket{00a_3}^{3}_{1}=\ket{001}$.
For a generic set of index $\Gamma=\set{k_1,\dots,k_l}$ and $\gamma\in\{0,1\}^l$ we write: \[ \ket{a}^{\Gamma}_{\gamma}=((1-\delta_{1\Gamma})\ket{a_1}+\delta_{1\Gamma}\ket{\gamma}^\Gamma_1)\otimes\dots\otimes ((1-\delta_{n\Gamma})\ket{a_n}+\delta_{n\Gamma}\ket{\gamma}^\Gamma_n) \]
For example, if $\Gamma=\set{1,3,7}$, $\gamma=(1,0,1)$ and $\ket{a}=\ket{00000000}$ then \[ \ket{a}^{\Gamma}_{\gamma}=\ket{10000010}. \]
