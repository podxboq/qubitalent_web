Gates

A single-qubit gate $\text{A}$ can be applied over the first qubit of a ket $\ket{a_1 a_2} $ and the result is $\text{A}\ket{a_1}\otimes \ket{a_2}$. It can also be applied over the last of both qubits and then the result is $\ket{a_1}\otimes \text{A}\ket{a_2}$.
Let $\text{A}$ be a single-qubit gate and $\ket{a}=\ket{a_1\cdots a_n}$ is a ket with $n$ qubits, the following equivalent notation is used: \[ \bigotimes_{k=1}^n\text{A}^{\delta_{hk}}\ket{a_k}= \begin{cases} \ket{a_1\cdots a_h\text{A}a_{h+1}\cdots a_n}\\ \ket{a}\text{A}_h\\ \ket{a}^{h}_{\text{A}} \end{cases} \]
For example, if $\ket{a}=\ket{a_1 a_2}$ then \[ \text{A}\ket{a_1}\otimes \ket{a_2} = \begin{cases} \ket{a_1\text{A}a_2}\\ \ket{a}\text{A}_1\\ \ket{a}^{1}_{\text{A}} \end{cases}. \]
Let $\Gamma=\set{k_1,\dots,k_l}\subseteq\mathbb{N}_n$ be a set. Let $\text{A}$ be a single-qubit gate, the following notation is used: \begin{align} \text{A}_{\Gamma} &= \text{A}_{k_1,\dots,k_l}=\prod_{k\in\Gamma}\text{A}_k \end{align}
Note that, according to the previous notation, for the specific case of the empty set $\Gamma=\emptyset$, the resulting quantum gate is the identity gate $\text{A}_\emptyset=\text{I}$. For example, if $\ket{a}=\ket{a_1 a_2}$ and $\Gamma = \set{1,2}$, then \[ \text{A}\ket{a_1}\otimes \text{A}\ket{a_2} = \begin{cases} \ket{a_1\text{A}a_2\text{A}}\\ \ket{a}\text{A}_{1,2}=\ket{a}\text{A}_{\Gamma}\\ \ket{a}^{1,2}_{\text{A}}=\ket{a}^{\Gamma}_{\text{A}} \end{cases}. \]
